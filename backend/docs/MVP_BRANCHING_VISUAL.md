## 🔑 Key Takeaways for Implementation

### 1. **Religion is the Master Branch**
- Muslim users: Skip ALL caste questions (Q22-Q24, Q27)
- Hindu/Jain/Sikh: Get full caste flow (up to 5 extra questions)
- Diet options change: "Halal only" appears for Muslims, "Jain food" for Jains

### 2. **Geography Doubles Up**
- NRI users get 3 extra location questions (Q11, Q17, Q34)
- India-based users get state question (Q12)
- These never overlap (mutually exclusive)

### 3. **Life Stage Matters**
- "Never married" = skip children existing (Q5)
- "Divorced/Widowed" = ask about existing children
- This affects ~20% of user base

### 4. **Children Intent Cascade**
- Only ask timeline (Q67) if user wants children
- Prevents awkward "N/A" answers

### 5. **Two-Way Filtering**
- Many questions have matching preference questions
- Example: Q47 (Your smoking) ↔ Q48 (Partner smoking OK?)
- Must check BOTH directions for compatibility

## 🎯 UX Recommendations

1. **Show dynamic progress:** "34 of ~73 questions" (changes based on path)

3. **Visual branching:** "✓ Skipped 4 questions (not applicable to Muslims)"
4. **Preview match pool:** After Tier 1, show: "X profiles match your criteria so far"
